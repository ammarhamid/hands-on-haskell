Preliminaties

Start GHCi:

Prelude>
Supports command line editing (readline).

Numbers

> > 1
> 1
> > 1 + 1
> 2
> > 2 * (3.1 - 1)
> 4.2
Also: (-), div, mod, (/), …
Q: what is 56088 divided by 456?

Bindings

> > let x = 1
> > x + x
> 2
GHCi only; In a file, just use x = 1.
Non-mutable; Re-binding shadows.

Files

> x = 1
> y = x + 2

Save this with extension .hs.

Load in GHCi with ghci <filename>

Reload with :r

Booleans

> > True
> True
> > not True
> False
> > True && (False || True)
> True

Application

> > not True
> False
Application with space.
> > min 1 2
> 1
Multiple arguments with another space.

If-then-else

> > if True then 1 else 2
> 1

This is an expression:
you cannot leave out a branch.

C.f. ternary operator.

Types

> > :t True
> True :: Bool
> > :t not
> not :: Bool -> Bool

Type classes

> > :t 1
> 1 :: Num a => a

Num is a type class.

Says: 1 can be any type that is numeric.

Characters and strings

> > :t 'c'
> 'c' :: Char
> > :t "Hello world"
> "Hello World" :: String

> "FPdays"
Q: What is the type of "Hello World" really?

Lists

> > :t [True, True, False]
> [True, True, False] :: [Bool]
> > :t []
> [] :: [a]
> > 1 : [2,3]
> [1,2,3]
> > [1,2,3] ++ [4,5,6]
> [1,2,3,4,5,6]
Q: Try to make a list
containing one, two and false.

List functions

> > length [1,4,7]
> 3
    
> > take 3 [1..10]
> [1,2,3]
    
> > drop 3 [10,9..1]
> [7,6,5,4,3,2,1]

List functions - 2

> > head [4,5,6]
> 4
    
> > tail [4,5,6]
> [5,6]
    
> > elem 3 [1,2,4]
> False

Infinite lists

> > head [1..]
> 1
    
> > take 10 (cycle [1,2,3])
> [1,2,3,1,2,3,1,2,3,1]
    
> > head (tail (repeat 2))
> 2

Equality

> > 1 == 1
> True
> > [1,2,3] /= [1,2,4]
> True

Works on most types.

Q: what is the type of (==)?

Comparison

> > 1 > 2
> False
> > [1,2,3] < [1,2,4]
> True
Also (<=), (>=).
> > :t (<)
> (<) :: Ord a => a -> a -> Bool
